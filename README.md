## Description
This laboratory exercise focuses on the advanced queries of MySQL. 
It covers the use of built-in string functions such as UPPER(), 
REPLACE(), CONCAT(), and REPEAT(). It also includes numeric functions 
like AVG(), and date functions like DATE_FORMAT(). 
Additionally, the exercise demonstrates the use of GROUP BY to group 
results, ORDER BY to sort records alphabetically, and constraints such 
as PRIMARY KEY, NOT NULL, AUTO_INCREMENT, and COMMENT in table creation.
